Pascal's Triangle

Description

By Eddie Shore, January 2016 (Eddie's Math and Calculator Blog)

Calculates the values of the rows of Pascal's_triangle

To get values of row n:
Enter row number n
f A to run program
R/S to get each row element

Examples:
Row 4 (n = 4): 1, 4, 6, 4, 1
Row 8 (n = 8): 1, 8, 28, 56, 70, 56, 28, 8, 1
