Solve a system of linear equations

Description

Random version :Up to 7 equations, each with 7 coefficients and 1 constant

N GSB 1 where N is the number of equations (less than 8).

A N N is displayed.
Matrix [A] has a dimension of NxN.

Enter the coefficients of x.
Press R/S after each is entered.

B N 1 is displayed.
Matrix [B] has a dimension of Nx1.

Enter the constants.
Press R/S after each is entered.

C N 1 is displayed.
Matrix [C] has a dimension of Nx1.

Values for x are read out in turn.

Lines 10 and 20 could be replaced with f R/S to allow manual data entry. 64 elements are allowed using a HP15C so [A] should be no greater than 7x7. [B] would then be 7x1 and [C] would be 7x1 for a total of 63 elements.

The simulator seems to be able to solve 8 equations, each with 8 coefficients and 1 constant. Including the 8x1 result matrix [C], this is 80 elements in memory.
