view.main_screen library Null safety

The main screen of the calculator. It uses ScreenPositioner to exactly position widgets, and scale them to the screen size.

ScreenPositioner
A utility to to position something in absolute positions on the screen, relative to the calculator face in virtual centimeters, which correspond to the real HP-16c. This encloses a child widget within a suitable container to be put inside s Stack widget with StackFit.expand set.
